# Bloom Mentor 技能进化总结

> 从提示词到专业 Skill 的完整进化历程

## 📋 概述

**技能名称**：Bloom Socratic Learning Mentor（Bloom Mentor）
**当前版本**：v2.0
**核心定位**：基于 Bloom 分类学、2 Sigma 一对一辅导和掌握学习法的 Socratic AI 导师

---

## 🎯 进化方向与目标

### 进化方向

```
阶段1：基础提示词
    ↓
阶段2：结构化提示词
    ↓
阶段3：初级 Skill（掌握学习 + Socratic 对话）
    ↓
阶段4：专业 Skill（八维概念解剖 + 完整学习系统）
```

### 核心目标

1. **从灌输到引导**：通过 Socratic 提问而非直接灌输，帮助用户构建真正的理解
2. **从模糊到结构**：基于 Bloom 分类学的6个认知层级，提供清晰的学习路径
3. **从被动到主动**：培养用户的元认知能力和自主学习策略
4. **从零散到系统**：提供完整的学习进度跟踪和效果评估

## 🎨 应用场景

### 场景 1：系统化学习新技术
**案例**：学习 Python 编程
- 从基础概念的记忆层开始
- 通过 Socratic 对话逐步深入
- 达到应用层时能独立完成实际编程任务

### 场景 2：深度理解复杂概念
**案例**：理解机器学习
- 使用八维概念解剖全面分析概念
- 从历史、发展、应用等多角度构建理解
- 通过掌握验证确保真正理解

### 场景 3：构建企业级 AI 应用
**案例**：学习 harness 工程
- 结合用户实际问题（模型"瞎回答"）
- 通过层级约束和提示词工程解决
- 构建完整的 harness 工程知识体系

### 场景 4：培养元认知能力
**适用人群**：希望提升学习能力的学习者
- 学习如何学习
- 掌握有效的学习策略
- 建立长期学习能力

## 🚀 使用方法

### 在 Claude Code / Trae 中使用

1. **唤醒技能**：提及学习需求
2. **描述目标**：告诉导师想学什么
3. **参与对话**：回答引导式问题
4. **提供反馈**：分享理解和建议
5. **持续进阶**：基于反馈调整路径

### 新兴技术学习流程

1. **准备知识库**：收集相关资料和概念定义
2. **初始化**：描述学习目标和背景
3. **概念分析**：八维解剖构建全面理解
4. **Socratic 对话**：深度探索核心问题
5. **掌握验证**：确认达到学习目标
6. **进度保存**：记录学习轨迹
