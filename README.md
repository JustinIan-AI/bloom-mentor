# Bloom Socratic Learning Mentor

> 基于 Bloom 分类学、2 Sigma 一对一辅导和掌握学习法的 Socratic AI 导师

## 🎯 技能用途

Bloom Socratic Learning Mentor 是一个专业的 AI 学习导师技能，专为以下场景设计：

- **深度学习需求**：希望真正理解概念而非死记硬背
- **个性化学习**：需要根据个人水平和目标定制学习路径
- **复杂概念理解**：面对抽象或新兴技术概念需要全面分析
- **长期学习规划**：希望建立系统的学习体系和进度跟踪
- **元认知发展**：希望培养自主学习能力和学习策略

## 📚 学习方式

### 核心学习方法
- **Socratic 引导对话**：通过提问而非直接灌输，帮助构建自己的理解
- **掌握学习机制**：确保真正理解后再进入下一阶段
- **自适应学习路径**：根据反馈自动调整内容深度和方向
- **八维概念分析**：从多个维度全面理解复杂概念
- **进度跟踪系统**：完整记录学习轨迹，生成可视化仪表板

### 学习体验
- **对话式学习**：通过问答形式进行，自然流畅
- **循序渐进**：从基础到高级，确保每个阶段都真正掌握
- **个性化反馈**：基于您的回答提供针对性指导
- **深度分析**：对复杂概念进行多角度剖析
- **成果可视化**：通过学习仪表板查看进度和成就

## 🎓 什么是 Bloom 分类学习法

Bloom 分类学是由教育心理学家 Benjamin Bloom 提出的认知能力分类框架，将学习目标分为六个层级，从低到高逐步递进：

### 六个认知层级

1. **记忆 (Remember)**
   - 识别和记忆基本事实、概念和术语
   - 关键词：定义、列表、识别、回忆

2. **理解 (Understand)**
   - 解释概念，用自己的话表达
   - 关键词：解释、描述、总结、对比

3. **应用 (Apply)**
   - 将知识应用到新的情境中
   - 关键词：应用、执行、解决、演示

4. **分析 (Analyze)**
   - 分解概念，识别关系和结构
   - 关键词：分析、分解、比较、分类

5. **评估 (Evaluate)**
   - 基于标准进行判断和评价
   - 关键词：评估、判断、批评、验证

6. **创造 (Create)**
   - 综合知识，创造新的概念或产品
   - 关键词：创造、设计、发明、组合

### 学习方法特点

- **结构化**：清晰的学习路径，从基础到高级
- **循序渐进**：每个层级都有明确的学习目标
- **能力导向**：关注认知能力的发展，而非只是知识积累
- **全面性**：覆盖从记忆到创造的完整认知过程
- **可衡量**：每个层级都有明确的评估标准

## 🚀 技能使用方法

### 在 Claude Code 中使用

1. **唤醒技能**：在对话中提及学习需求
2. **开始学习**：描述您的学习目标和当前水平
3. **参与对话**：回答 Socratic 引导问题，构建理解
4. **提供反馈**：分享您的理解和困惑
5. **持续学习**：系统会基于您的反馈调整学习路径

### 在 Trae 中使用

1. **导入技能**：将 bloom-mentor 技能添加到您的 Trae 环境
2. **触发技能**：通过关键词或直接请求激活技能
3. **开始会话**：描述您想学习的主题
4. **互动学习**：参与引导式对话，完成掌握验证
5. **跟踪进度**：查看系统生成的学习仪表板

### 新兴技术学习流程

1. **准备知识库**：收集相关资料，整理核心概念
2. **初始化**：描述学习目标和背景
3. **概念分析**：八维解剖构建全面理解
4. **Socratic 对话**：深度探索核心问题
5. **掌握验证**：确认达到学习目标
6. **进度保存**：记录学习轨迹

## 💬 会话示例

### 示例 1：学习 Python 编程

**用户**：我想从零开始学习 Python 编程

**Bloom Mentor**：
> 很高兴帮你学习 Python 编程！让我们从基础开始。
>
> **开始评估**：
> 1. 你之前有编程经验吗？
> 2. 你学习 Python 的主要目标是什么？
> 3. 你希望达到什么水平？
>
> 基于你的回答，我会为你设计个性化的学习路径。

**用户**：我没有编程经验，想学习 Python 来做数据分析，希望能达到中级水平。

**Bloom Mentor**：
> 明白了，我们从 Bloom 分类学的记忆层开始。
>
> **概念分析**：
> 我们先对 Python 编程进行八维分析，帮助你建立全面理解。
>
> **记忆层问题**：
> - Python 中最基本的数据类型有哪些？
> - 什么是变量？如何在 Python 中定义变量？
> - Python 的基本操作符有哪些？
>
> 请用你自己的理解回答这些问题，不需要担心答错。

## ❓ 常见问题

### Q: 如何确定合适的学习起点？
**A**: 系统会通过 2-3 个诊断问题评估您的 prior knowledge，自动确定起始层级。

### Q: 学习进度存储在哪里？
**A**: 学习进度会存储在您指定的路径中，未指定则默认存储在当前工作目录。

### Q: 可以同时学习多个主题吗？
**A**: 可以，每个主题会在独立的目录中存储学习记录。

### Q: 如果我达到了最高层级怎么办？
**A**: 进入元认知反思阶段，提供综合创造任务和学习策略指导。

### Q: 如何查看我的学习进度？
**A**: 系统会生成学习仪表板，展示您的学习轨迹和掌握情况。
